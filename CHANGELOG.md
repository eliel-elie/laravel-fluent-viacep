# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## v1.0.0 - 2026-02-16

### Added
- Initial release
- Fluent interface for ViaCEP API
- Support for JSON, XML, Piped, and JSONP formats
- Address DTO for type safety
- Integrated caching support
- Automatic retry functionality
- Bulk request support
- CEP validation and formatting helpers
- Full test suite with Pest
